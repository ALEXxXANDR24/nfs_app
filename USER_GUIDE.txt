5. Работа с приложением

7. Решение проблем

========================
1. ТРЕБОВАНИЯ К СИСТЕМЕ

WINDOWS:
  - Windows 10 или выше
  - OpenVPN установлен (https://openvpn.net)
  - NFS Client компонент (устанавливается автоматически)

LINUX:
  - Ubuntu 18.04+ или аналогичный дистрибутив
  - OpenVPN установлен (устанавливается автоматически)
  - nfs-common установлен (устанавливается автоматически)

MAC OS:
  - macOS 10.12 или выше
  - OpenVPN (устанавливается автоматически)
  - Встроенная поддержка NFS

==================
2. ПЕРВЫЙ ЗАПУСК

WINDOWS:
  1. Запустите NFS Connect.exe
  2. При первом запуске приложение проверит NFS Client
  3. Если компонент не установлен:
     - Появится сообщение "NFS Client Required"
     - Приложение попросит разрешение администратора (UAC диалог)
     - Нажмите "Да" в диалоге Windows
     - Компонент установится автоматически

LINUX/MAC OS:
  1. Откройте терминал в папке приложения
  2. Выполните: python -m pip install -r requirements.txt
  3. Запустите: python nfs_vpn_app/main.py
  4. Приложение проверит зависимости автоматически

================
3. АВТОРИЗАЦИЯ

После установки компонентов откроется окно авторизации:

    Email:    Введите вашу почту HSE (example@edu.hse.ru)
    Password: Введите ваш пароль HSE

==============================
4. ПОДКЛЮЧЕНИЕ И МОНТИРОВАНИЕ

!!! После первого запуска необходимо перезагрузить устройства перед началом работы.

После успешной авторизации приложение автоматически:

   Проверяет подключение к VPN
      → Если VPN не подключен, автоматически подключается
      
   Подключается к серверу по SSH
      → Получает список доступных папок
      
   Монтирует NFS диск
      → На Windows: виртуальный диск (напр., D: или E:)
      → На Linux: папка /mnt/nfs (требует пароль для sudo)
      → На Mac: папка в Home (~/ nfs)

Процесс занимает 15-30 секунд.

На Linux/Mac может потребоваться ввод пароля администратора:
  - В консоли появится приглашение: [sudo] password for username:
  - Введите пароль вашего компьютера

========================
5. РАБОТА С ПРИЛОЖЕНИЕМ

После успешного монтирования откроется главное окно:

WINDOWS:
  Диск будет доступен в "Проводнике" (напр., D:\)
  Копируйте файлы через обычный файловый менеджер
  Создавайте ярлыки и работайте как с обычным диском

LINUX:
  Диск доступен в /mnt/nfs/
  Откройте: файловый менеджер → /mnt/nfs/
  Используйте команды: cp, mv, ls и т.д.

MAC OS:
  Диск доступен в ~/nfs/
  Откройте: Finder → перейдите в Home (~)
  Используйте обычные операции файлов

==================================
6. ОТКЛЮЧЕНИЕ И ДЕМОНТИРОВАНИЕ

Когда закончили работу:

  1. Закройте все файлы на смонтированном диске
  2. Нажмите "Disconnect" в приложении
     (или закройте приложение)
  3. Приложение автоматически:
     - Отмонтирует NFS диск
     - Отключится от VPN
     - Закроется корректно

Перед отключением:
  - Убедитесь, что все файлы синхронизированы
  - Не оставляйте открытыми файлы на NFS диске
  - В Windows закройте диск в Проводнике

=======================
7. РЕШЕНИЕ ПРОБЛЕМ


ПРОБЛЕМА: VPN не подключается
РЕШЕНИЕ:
   - Проверьте, установлен ли OpenVPN

ПРОБЛЕМА: NFS диск не монтируется (Linux/Mac)
РЕШЕНИЕ:
   - Проверьте, есть ли права на /mnt/ или домашнюю папку
   - Используйте: mount | grep nfs чтобы проверить статус

ПРОБЛЕМА: NFS Client не устанавливается (Windows)
РЕШЕНИЕ:
   - Запустите приложение как администратор (правый клик → Запуск от администратора)
   - Может потребоваться перезагрузка компьютера
   - Попробуйте установить вручную:
     1. Панель управления → Программы → Включение/отключение компонентов Windows
     2. Найдите "Services for NFS" и "Client for NFS"
     3. Отметьте оба пункта и нажмите OK
     4. Перезагрузите компьютер
